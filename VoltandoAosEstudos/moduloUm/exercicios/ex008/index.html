Principais formatações

Negrito / Destaque

Nesta frase teremos um termo em negrito usando a tag <b>, não semântica

Nesta frase, temos um termo em destaqueusando a tag <strong> tag semântica

Itálico / Ênfase

Nesta frase, temos um termo em Itálico usando a tag <i>, não semântica.

Nesta frase temos um termo em ênfase usando a tag <em> semântica.

Marcação de texto

Podemos criar uma marcação de texto usando a tag <mark>

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele dever ser lido mas não considerado

Texto Inserido

Podemos marcar um texto como inserido para dar mais ênfase e indicar que ele foi adicionado depois

Existe tambem o sublinhado com a tag <u> porém, não é semântica.
